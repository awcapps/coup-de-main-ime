# Screenshots

Ce dossier contient les screenshots/démos de Coup de Main pour le README et la documentation.

## À Ajouter

- [ ] Screenshot du clavier IME en action
- [ ] GIF de la notation d'un coup
- [ ] Screenshot des paramètres/customisation
- [ ] Screenshot de l'historique des parties
- [ ] Screenshot de l'app sur tablette

## Guidelines

- **Format** : PNG pour screenshots, GIF pour animations
- **Taille** : Max 1920px de largeur
- **Nommer** : `keyboard-view.png`, `settings-screen.png`, etc.
- **Optimiser** : Compresser avant commit (TinyPNG, etc.)

## Utilisation

Référencer dans README.md :
```markdown
![Clavier Coup de Main](docs/screenshots/keyboard-view.png)
```
